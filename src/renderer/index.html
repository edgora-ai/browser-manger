👋 Welcome to CloakLite

 
Let's get your fingerprint browser ready in 3 quick steps.


 

  

   
1 Install CloakBrowser

   
Downloads the open-source Chromium binary used by all profiles. Roughly ~150 MB. One-time download.

   

   

    
   

  


  

   
2 Create your first profile

   
A profile bundles a fingerprint seed, optional proxy, and isolated browser storage.

   

    
    
   

   

    
   

  


  

   
3 Launch & check fingerprint

   
Starts the profile and opens ping0.cc/env so you can verify fingerprint isolation.

   

    
   

  


  

   
4 Configure AI Agent (optional)

   
Wire up an OpenAI-compatible or Claude provider so you can drive browser automation with natural language.
